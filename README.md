# Web Chat created with React, Socket.io, Express and NodeJS
<br/>
- front-end: ./client
- back-end: ./server
<br />
# How to run
- # Client
- open a new terminal window in the source directory
- `cd client`
- `npm start`
- # Server
- open a new terminal window in the source directory
- `cd server`
- `npm start`
